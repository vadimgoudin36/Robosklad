Warehouse Robot Simulator
Java-приложение для моделирования автоматизированного склада

Описание
Модель работы автоматизированного складского комплекса с роботами-комплектовщиками. Проект демонстрирует применение принципов объектно-ориентированного программирования для решения прикладной задачи из области логистики.

Основные функции
Визуализация склада – графическое отображение стеллажей и роботов

Автоматическое управление – роботы самостоятельно перемещаются к стеллажам

Учет ресурсов – отслеживание заряда батарей и количества груза

Работа с заказами – создание и просмотр заказов на сборку

Пошаговая симуляция – выполнение одного шага работы склада

Технологии
Язык: Java 11+

Интерфейс: JavaFX

Архитектура: MVC (Model-View-Controller)

Сборка: Maven

Запуск
Установите Java JDK 11+ и JavaFX SDK

Клонируйте репозиторий:

bash
git clone https://github.com/ваш-логин/warehouse-simulator.git
Импортируйте проект в IDE (IntelliJ IDEA, Eclipse)

Настройте путь к JavaFX SDK

Запустите класс Main

Использование
Запустите приложение

Нажмите "Назначить и запустить" для старта симуляции

Наблюдайте за перемещением роботов (синие круги) к стеллажам (зеленые/серые прямоугольники)

Выбирайте заказы в списке для просмотра их состава

Наводите курсор на объекты для детальной информации
