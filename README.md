# Warehouse Robot Simulator

**Java-приложение для моделирования автоматизированного склада**

---

## Описание

Объектно-ориентированная модель автоматизированного складского комплекса с роботами-комплектовщиками. Проект демонстрирует применение принципов ООП для решения прикладной задачи из области логистики и автоматизации.

## Основные функции

- **Визуализация склада** – графическое отображение стеллажей и роботов на координатной плоскости
- **Автоматическое управление** – роботы самостоятельно находят и перемещаются к ближайшим стеллажам
- **Учет ресурсов** – отслеживание уровня заряда батарей и количества груза на стеллажах
- **Работа с заказами** – создание и просмотр заказов на сборку
- **Пошаговая симуляция** – выполнение одного шага работы склада с обновлением всех состояний

## Технологический стек

- **Язык программирования:** Java 11+
- **Графический фреймворк:** JavaFX
- **Архитектура:** MVC (Model-View-Controller)
- **Система сборки:** Maven
